# WEDE5020__CycleHub__P3
CycleHub Website
Project Overview

CycleHub is a modern, interactive website designed for a bicycle shop. It provides users with a seamless experience for browsing bikes, viewing images, and contacting the store. The project focuses on responsive design, user interactivity, and enhanced visuals, transforming a basic static HTML website into a professional, user-friendly platform.

Features
1. Product Showcase
Bike Cards: Display bikes with images, names, and prices.

Modals: Click on any bike to open a modal with detailed specifications and additional images.

Gallery Lightbox: Interactive panel to scroll through multiple images for each bike.

2. Navigation & Layout

Responsive navigation bar linking all pages.

Consistent layout across all five pages using semantic HTML elements (<header>, <nav>, <main>, <footer>).

Modern design with color schemes, shadows, and rounded corners.

3. Interactivity & Animations

Smooth hover effects and modal animations.

Drag-and-drop and dynamic animations enhance user engagement.

Responsive behavior ensures optimal viewing on different devices.

4. Contact & Location

Interactive map added on the Contact Us page for store location.

Users can easily locate the shop and reach out for inquiries.

5. Accessibility & SEO

Proper use of title tags, meta descriptions, and alt attributes for images.

Clean, readable typography and color contrast for better accessibility.

Major Updates

Fixed issues with Cannondale bike modal not opening.

Added a Gallery Lightbox for bike images.

Introduced animations and drag-and-drop elements across multiple pages.

Added interactive maps to Contact Us page.

Updated overall styling and responsiveness using CSS and media queries.

Technologies Used

HTML5 & CSS3: Structure, layout, styling, and responsiveness.

JavaScript: Modal functionality, gallery interactivity, animations, and user feedback.

CSS Grid & Flexbox: Responsive layouts for galleries and cards.

Google Maps / Embedded Map: Interactive store location.
